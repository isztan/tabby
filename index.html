Basic Tabs

These use basic, inline hyperlinks.

Tabby matches the href value on each link to the id of the matching .tabs-pane content. Add the .active class to the tab content you want to display by default.

Superheroes | Ice Cream | Seasons

Superheros

Spiderman, Batman, or Iron Man... which one is your favorite?

Ice Cream

Chocolate, vanilla or strawberry?

Seasons

Stacked Navigation

On smaller screens, stacked tabs mimic the appearance of horizontal tabs. On bigger screens, navigation appears in a vertical stack next to the content.

This demo uses Origami, a fluid, mobile-first grid system, to align the navigation. Feel free to use whatever grid you prefer.

Progressively Enhanced

By default, tab navigation is hidden and all tab content is visible.

Tabby includes the JS Accessibility snippet, which adds a .js class to the <body> on page load. This activates Tabby, showing the tab navigation and hiding tab content.

Requires jQuery

Tabby requires jQuery.
